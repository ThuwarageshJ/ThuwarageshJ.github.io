Every now and then I come across an idea, a project, or a thought that I want to remember —
or share with someone. Bookmarks pile up, notes get lost, and conversations forget themselves.

So I made this site. Nothing fancy. Just a place to put things down.

## What you'll find here

- **Projects & Research** — things I've built or studied
- **Thoughts** — longer pieces, like what you're reading now
- **Ideas** — half-formed sparks that might turn into something

## How this works

I write blog posts in plain Markdown, drop them into the `blogs/` folder, and they show up here.
If I want to add an image, I just drop it in the `images/` folder and reference it:

```markdown
![a cool diagram](images/diagram.png)
```

That's it. No build tools, no CMS, no databases. Just files on GitHub Pages.

## Why Markdown?

Because it gets out of the way. I can focus on writing instead of fiddling with formatting.
And when I need structure — headings, lists, code blocks, images — it's all there.

> "The best tool is the one that disappears while you're using it."

Anyway, thanks for stopping by. More to come.
